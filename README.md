# MO2 Plugins Collection

A collection of plugins for **Mod Organizer 2 (MO2)** that add quality-of-life improvements and additional functionality.

---

## Install Multiple Mods
Allows installing multiple mods at the same time instead of one-by-one.

**Nexus:**  
https://www.nexusmods.com/site/mods/798

---

## Mod Auto-Installer
Automatically installs mods on download completion.

**Nexus:**  
https://www.nexusmods.com/site/mods/799

---

## Undo Move
Adds the ability to undo mod move operations.

**Nexus:**  
https://www.nexusmods.com/site/mods/809

---

## Query All Info
Adds a button to query all missing meta information for downloads.

**Nexus:**  
https://www.nexusmods.com/site/mods/1730

---

## Pause or Resume All Downloads
Adds buttons to pause or resume all downloads at once.

**Nexus:**  
https://www.nexusmods.com/site/mods/1731
